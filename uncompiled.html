vector-06js

Online Vector-06c emulator © 2016 Viacheslav Slavinsky, caglrc.cc
Standalone version
Useful Keys:
F11 reboot
F12 restart (press F12 to launch floppy image)
Alt ПС
Shift СС
Ctrl УС
Shift+Bksp СТР
F6 РУС/ЛАТ
Single ROM files are launched automatically. R0M files are loaded at address $0000. FDD images are mounted in drive A and booted using a standard boot ROM. To start the program after the floppy has booted, press F12.
If the ware is a zip archive, a file chooser will ask you to pick one file.
It is also possible to build a floppy image from multiple files, this is useful to be able to run programs that consist of several files but are archived in separate files instead of disk images.
Click into emulator window to switch to fullscreen.
The "standalone" version of this emulator supports uploading ROM and floppy images from the client pc. Check it out.
